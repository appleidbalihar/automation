# Workflow

1. Research feature across `apps` & `packages`.
2. Modify or add domain types in `packages/contracts`.
3. Update backend in Fastify (`api-gateway` or `workflow-service` or `packages/db`).
4. Update frontend React hooks and `ui-kit` / ReactFlow items in `apps/web`.
5. Write Tests and check formatting via `pnpm lint` && `pnpm build`.
6. Add Developer and Operations Documentation under `docs/`.
